    extract_tags(tag_string) click to toggle source

    Extracts the tags from a tag string. If a level is provided during initialization the level is restricting the count of tags imported based on the level value

    # File lib/syclink/importer.rb, line 47
def extract_tags(tag_string)
  if tag_string.empty? || opts[:level] == 0
    "Default"
  else
    tags = tag_string.first.split('/')
    level = [opts[:level] || tags.size, tags.size].min
    tags[-level..-1].join(',')
  end
end
